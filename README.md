# 🚉 えきBoard (EkiBoard) へようこそ！

スマホが「駅の電光掲示板」に早変わり！？✨
「**えきBoard**」は、あなたの今いる場所から1番近い駅を探し出して、駅にあるあの「パタパタ動く電光掲示板（発車標）」を画面に再現しちゃうアプリです！🚀

---

## 🌟 アプリのすごいところ (特徴)

- **📍 今すぐ最寄り駅がわかる！**
  アプリを開くだけで、GPSを使って今いる場所の最寄り駅を自動でキャッチ！すぐに見たい路線の時間が分かります。
- **🗺️ 地図から気になる駅をサクッと検索！**
  「あの駅の時間はどうなってるんだろう？」って時も大丈夫！検索機能や地図から、全国の駅をカンタンに探せます🔍
- **⭐ いつもの駅を「お気に入り」に！**
  毎日使う駅や路線は、星マークをタップして「お気に入り」に登録！いつでもすぐにチェックできます💖
- **🚉 「当駅始発」でゆったり座ろう！**
  ボタンひとつで、その駅から出発する「始発列車」だけを表示！「今日は座って帰りたいな…」って時にめちゃくちゃ便利です💺
- **🌐 4カ国語に対応！**
  日本語・英語・中国語（簡体字）・韓国語に対応！設定画面からワンタップで切り替わって、言語の勉強や外国の人への案内にも便利です🌍
- **📱 リアルな「パタパタ」UI！**
  画面をタップすると、3Dで「パタパタッ」と上り方面・下り方面が切り替わるよ！本物みたいでちょっと楽しい😆

---

## 💡 基本的な使い方

1. **アプリを開く！**
   すぐに最寄り駅の電光掲示板が表示されます。
2. **掲示板をタップ！**
   上り線・下り線がパタパタと入れ替わります🔄
3. **地図（検索タブ）で遊ぶ！**
   地図を動かして駅のピンをタップしたり、名前で駅を検索してみてね。
4. **星マークをポチッ！**
   よく使う路線で「⭐」を押せば、お気に入りタブに自分だけの掲示板リストが作れます！
